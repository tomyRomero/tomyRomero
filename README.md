## Getting Started

Instructions on how to clone the repository and set up the project locally.

```bash
# Clone the repository
git clone https://github.com/tomyRomero/tomyRomero

# Navigate to the project directory
cd tomyRomero

# Install dependencies
npm install

# Run project
npm run dev

```
